## How to Add More Cities

To expand coverage, edit `lib/us-states.ts`:

```typescript
export const MAJOR_CITIES: Record<string, City[]> = {
  STATE_CODE: [
    { name: 'City Name', lat: latitude, lon: longitude },
    // Add more cities here
  ],
};
```
